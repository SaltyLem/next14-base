## Features

アプリケーション内の各機能を独立したモジュールとして分離し関連するコンポーネント、フック、スタイル、ユーティリティ関数などを配置します。

## Example

```
features/
├── blog/
│   ├── components/
│   │   ├── BlogPost.ts
│   │   ├── BlogList.ts
│   ├── hooks/
│   │   ├── useFetchPosts.ts
│   ├── styles/
│   │   ├── blog.module.css
│   ├── utils/
│   │   ├── formatDate.ts
│   ├── index.ts
```
